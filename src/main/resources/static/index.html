Objetivo do Projeto

O objetivo do GerTask é oferecer uma plataforma moderna para gerenciamento de tarefas pessoais e projetos, utilizando uma interface intuitiva inspirada em metodologias ágeis.

O sistema foi desenvolvido para unir produtividade, organização visual e segurança, permitindo que usuários criem tarefas, acompanhem progresso e mantenham informações protegidas.

Tecnologias Utilizadas

O backend foi construído utilizando Spring Boot, Spring Security, JWT, JPA/Hibernate e MySQL.

A aplicação segue padrões modernos de arquitetura REST, DTOs, tratamento global de exceções, Swagger/OpenAPI e autenticação segura.
